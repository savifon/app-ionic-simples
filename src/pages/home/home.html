Seja bem-vindo ao App Simples IONIC!

O objetivo é fazer um app simples alimentando os dados a partir de uma API. Neste caso usaremos a The Movie DB.

Clique em FEED e veja os cards cujo dados são consumidos da API.

Não esqueça de gerar sua API Key no site The Movie DB e inserir na página movie.ts para o FEED carregar corretamente.
